Cabecera (HEAD) de un documento HTML

La HEAD es la primera de las dos partes en que estructuran un documento HTML.
En la HEAD reside información acerca del documento, y generalmente no se ve cuando se navega por él. En la HEAD se pone el elemento lleno < TITLE>.
No hay que confundir el elemento < TITLE> con el nombre del fichero. Por ejemplo, esta página la e creado en un fichero llamado manualmarcas.html y el texto de su < TITLE> es: Manual de HTML.
Se escibirá asi:
< HEAD>
< TITLE>Head de un documento < /TITLE>
< /HEAD>

Dentro de la HEAD puede utilizarse otro elemento: META. Por ejemplo, si se escribe: < HEAD>
< TITLE>Head de un documento < /TITLE>
< META HTTP-EQUIV="Refresh" CONTENT="10">
< /HEAD>
Esto hace que el visualizador vuelva a cargar la página activa al cabo de 10 segundos.

Tamaños y tipos de letra en HTML

Para definir distintos tamaños de letra, en HTML se utiliza el elemento < H "numero del 1 al 6"> siendo 1 el tamaño mayor.
Se escribirán así:

< H1> Texto de prueba (H1)< /H1>
< H2> Texto de prueba (H2)< /H2>
< H3> Texto de prueba (H3)< /H3>
< H4> Texto de prueba (H4)< /H4>
< H5> Texto de prueba (H5)< /H5>
< H6> Texto de prueba (H6)< /H6>

y este sería el resultado:

Texto de prueba (H1)

Texto de prueba (H2)

Texto de prueba (H3)

Texto de prueba (H4)

Texto de prueba (H5)
Texto de prueba (H6)

Otra forma de cambiar los tamaños de letra es utilizar el elemento con el atributo VALOR, que es un número entre 1 y 7.
Se escribiría así:
< FONT SIZE=3>A< /font>< FONT SIZE=4>A< /font>< FONT SIZE=5>A< /font>
< FONT SIZE=6>A< /font>< FONT SIZE=7>A< /font>< FONT SIZE=6>A< /font>
< FONT SIZE=5>A< /font>< FONT SIZE=4>A< /font>< FONT SIZE=3>A< /FONT>

Dará como resultado:
AAA AAA AAA

Parrafos

Para definir los párrafos se utiliza el elemento lleno < P> < /P>.
Se escribirá así:

< P> Texto 1 Texto 1 Texto 1 Texto 1 Texto 1 Texto 1 Texto 1 Texto 1 < /P>
< P> Texto 2 Texto 2 Texto 2 Texto 2 Texto 2 Texto 2 Texto 2 Texto 2 < /P>

Y este sería el resultado:

Texto 1 Texto 1 Texto 1 Texto 1 Texto 1 Texto 1 Texto 1 Texto 1

Texto 2 Texto 2 Texto 2 Texto 2 Texto 2 Texto 2 Texto 2 Texto 2

Textos preformateados

Al escribir en HTML y queremos poner dos espacios entre palabras o alinear frases presionando el botón enter nos damos cuenta de que nos muestra todo con un solo espacio o todo junto, este efecto lo podemos eliminar con el elemento < PRE>.
Al texto que va dentro del elemento PRE se le pueden aplicar todos los elementos que se quiera, siempre que sean elementos que no alteren la posición del texto.
Se escribirá así: < PRE>
Este texto empieza aquí
continua por aquí
y finaliza aquí.
< /PRE>
y este sería el resultado:

       Este texto empieza aquí    

                              continua por aquí

                                                y finaliza aquí.

Listas

Hay elementos que permiten crear texto con varios formatos de listado: Estos pueden ser ordenados < OL> , desordenados < UL> , lista de definicion< DL>.

Para escribir una lista ordenada con números utilizaremos el elemento < OL>< /OL y por cada linea pondremos el elemento < LI>:
< OL>
< LI>Primera linea
< LI>Segunda linea
< /OL>
Se vería así:

  1. Primera linea
  2. Segunda linea

Para escribir una lista desordenada utilizaremos el elemento < UL>< /UL>, y por cada línea como en el anterior elemento utilizaremos el elemento < LI>.
Se escribe así:
< UL>
< LI>Primera linea
< LI>Segunda linea
< /UL>
Y se vería así:

  • Primera linea
  • Segunda linea

Para crear una lista de definición utilizaremos el elemento < DL>< /DL>, y en cada linea podemos poner dos elementos, el elemento < DT>,para que la linea empiece al principio, y el elemento < DD> para que el texto empiece mas adelantado.
Se escribiría así:
< DL>
< DT>Primera linea
< DD>Segunda linea
< /DL>

Y se vería así:

Texto con el elemento (DT)
Texto con el elemento (DD)
